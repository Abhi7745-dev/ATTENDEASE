# 🎓 AttendEase

AttendEase is a secure QR Code-based Attendance Management System developed to simplify and secure the attendance process in educational institutions. The system ensures that only students physically present in the classroom can mark their attendance using QR code scanning and GPS location verification.

---

#  Features

##  Professor

- Secure Login
- Generate QR Code for Attendance Sessions
- GPS-Based Session Creation
- Automatic Session Management
- Generate Attendance Reports
- Send Attendance Reports via Email
- Session-wise Attendance Records

---

##  Student

- Secure Login
- QR Code Scanning
- GPS Location Verification
- Instant Attendance Marking
- Duplicate Attendance Prevention

#  Attendance Workflow

1. Professor logs in and creates an attendance session.
2. QR Code is generated for the session.
3. Student scans the QR Code.
4. Student's GPS location is verified.
5. Attendance is recorded in the database.
6. Professor generates and receives the attendance report via email.
